## Data & Analytics > DataFlow > Overview

* A service that makes it easy to create the desired ETL (extract-transform-load) flows through drag & drop and simple configuration.
* Provides encryption filters and parsing filters that can perform simple transformations on input messages.
* Provides the ability to run flows periodically at desired times.

## Benefits

* Supports easy integration with NHN Cloud services.
* Enables safe and easy infrastructure resource management that reduces user burden.
* Plans to strengthen integration with various Data Platform and AI Service to be added in the future.
    * It will be possible to utilize it in various ways, such as business analysis and AI service development.

## Main Features

* You can create a flow that transforms data from original data sources through filters and delivers it to the desired data source.
* You can easily set up flow logic through drag & drop and simple configuration.
* Supports easy integration with other NHN Cloud services.
* Through the scheduling function, you can run flows at desired times.
