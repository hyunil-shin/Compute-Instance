## Data & Analytics > DataFlow > Overview

* A service that allows you to easily create desired ETL (extract-transform-load) flows through drag & drop and simple configuration.
* Provides encryption filters, parsing filters, and other tools that can perform simple transformation tasks on input messages.
* Provides the ability to run flows periodically at desired times.

## Benefits

* Supports easy integration with NHN Cloud services.
* Enables safe and easy infrastructure resource management with reduced user burden.
* Plans to strengthen integration with various Data Platform and AI Services to be added in the future.
    * Will be available for various applications including business analysis and AI service development.

## Key Features

* You can create flows that transform data ingested from original data sources through filters and deliver it to desired data sources.
* Flow logic can be easily configured through drag & drop and simple settings.
* Supports easy integration with other products within NHN Cloud.
* Flows can be executed at desired times through scheduling functionality.
