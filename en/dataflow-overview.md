## Data & Analytics > DataFlow > Overview

* A service that lets you easily create ETL (extract-transform-load) flows with drag & drop and simple configuration.
* Provides encryption filters, parsing filters, and more to perform simple transformations on input messages.
* Provides the ability to run flows periodically at desired times.

## Benefits

* Supports easy integration with NHN Cloud services.
* Enables safe and easy infrastructure resource management, reducing the burden on users.
* Plans are underway to strengthen integration with various Data Platforms and AI Services to be added in the future.
    * It will be applicable in many areas, including business analysis and AI service development.

## Key Features

* You can create flows that transform data ingested from original data sources through filters and deliver it to desired data sources.
* Flow logic can be easily configured through drag & drop and simple settings.
* Supports easy integration with other products within NHN Cloud.
* The scheduling feature allows you to run flows at desired times.
