## Data & Analytics > DataFlow > Overview

* A service that allows you to easily create desired ETL (extract-transform-load) flows through Drag & Drop and simple configuration.
* Provides encryption filters and parsing filters that can perform simple transformation tasks on input messages.
* Provides the ability to run flows periodically at desired times.

## Key Features

* Supports easy integration with NHN Cloud services.
* Enables safe and easy infrastructure resource management that reduces user burden.
* Plans to strengthen integration with various Data Platforms and AI Services in the future.
    * Will be available for use in various areas such as business analysis and AI service development.

## Main Features

* You can create flows that transform data received from original data sources through filters and deliver them to desired data sources.
* Flow logic can be easily configured through Drag & Drop and simple settings.
* Supports easy integration with other products within NHN Cloud.
* Through scheduling functionality, you can run flows at desired times.
