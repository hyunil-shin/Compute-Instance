## Data & Analytics > DataFlow > Overview

* This is a service that allows you to easily create desired ETL (extract-transform-load) flows through drag & drop and simple configuration.
* It provides encryption filters and parsing filters that can perform simple transformation tasks on input messages.
* It provides the ability to periodically execute flows at desired times.

## Benefits

* Supports easy integration with NHN Cloud services.
* Enables safe and convenient infrastructure resource management that reduces user burden.
* Plans to strengthen integration with various Data Platform and AI Services to be added in the future.
    * It will be available for various applications such as business analysis and AI service development.

## Key Features

* You can create flows that transform data ingested from original data sources through filters and deliver it to desired data sources.
* Flow logic can be easily configured through drag & drop and simple settings.
* Supports easy integration with other services within NHN Cloud.
* Flows can be executed at desired times through scheduling functionality.
