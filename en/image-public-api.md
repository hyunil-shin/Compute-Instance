## Compute > Image > API v2 Guide

To use the API, you need the API endpoint, token, and other information. See [Preparing to Use the API](/Compute/Compute/en/identity-api/) to prepare the information required for API usage.

The Image API uses the `image` type endpoint. For the exact endpoint, refer to `serviceCatalog` in the token issuance response.

| Type | Region | Endpoint |
|---|---|---|
| image | Korea (Pangyo) region<br>Korea (Pyeongchon) region<br>Japan region<br>US (California) region | https://kr1-api-image-infrastructure.nhncloudservice.com<br>https://kr2-api-image-infrastructure.nhncloudservice.com<br>https://jp1-api-image-infrastructure.nhncloudservice.com<br>https://us1-api-image-infrastructure.nhncloudservice.com |

Fields not specified in this guide may appear in API responses. These fields are used internally by NHN Cloud and are subject to change without prior notice, so do not use them.
