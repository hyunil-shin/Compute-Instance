## Security > Secure Key Manager > リリースノート

### 2026. 04. 14.
#### 機能改善/変更
  * `APPROVAL MEMBER`役割削除
    * Secure Key Manager APPROVAL MEMBER役割をSecure Key Manager ADMIN役割に移行し、役割体系を簡素化
  * 権限の細分化
    * `SecureKeyManager:API.ADMIN`、`SecureKeyManager:API.VIEWER`権限を追加し、コンソールおよびAPI権限を細分化して管理するよう変更

### 2026. 03. 10.
#### 新機能追加
  * API v1.3追加
    * `X-NHN-AUTHORIZATION`ヘッダーを通じたトークン認証方式を追加。詳細については、[API v1.3ガイド](/Security/Secure%20Key%20Manager/ja/api-guide-v1.3/)を参照。
  * 機密データ修正API追加 (v1.2、v1.3)
    * APIを利用してSecure Key Managerに保存した機密データを修正できる機能を追加。詳細については、[API v1.2ガイド](/Security/Secure%20Key%20Manager/ja/api-guide-v1.2/)または[API v1.3ガイド](/Security/Secure%20Key%20Manager/ja/api-guide-v1.3/)を参照。

### 2026. 02. 10.
#### 新機能追加
  * キーストア一覧詳細照会API追加
    * APIを利用してキーストアの詳細情報一覧を照会できる機能を追加。詳細については、[API v1.0ガイド](/Security/Secure%20Key%20Manager/ja/api-guide-v1.0/)または[API v1.2ガイド](/Security/Secure%20Key%20Manager/ja/api-guide-v1.2/)を参照。
  * キー一覧詳細照会API追加
    * APIを利用してキーの詳細情報一覧を照会できる機能を追加。詳細については、[API v1.0ガイド](/Security/Secure%20Key%20Manager/ja/api-guide-v1.0/)または[API v1.2ガイド](/Security/Secure%20Key%20Manager/ja/api-guide-v1.2/)を参照。

  * キー一覧/詳細照会API追加
    * APIを利用してキーID一覧およびキーIDを通じてキーを詳細照会できる機能を追加
  * 認証情報一覧/詳細照会API追加
